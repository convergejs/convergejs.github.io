RuntimeFlags enum

Represents a collection of runtime flags.

Signature:

export declare enum RuntimeFlags

Enumeration Members

Member Value Description
Any 7 Any runtime.
Electron 4 An Electron based runtime.
Node 2 A Node.js runtime.
NodeIntegrated 6 A runtime that is based on Node.js, or a runtime that has Node.js integrated as part of it.
Unknown 0 Information about the runtime is unknown.
Web 1 A browser based runtime.
