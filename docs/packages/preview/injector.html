injector package

Provides the types and utilities that support 'functional' and type based dependency injection.

Classes

Class Description
AmbiguousExportsError The error that is thrown when multiple exports exist for a given ExportDeclaration, and there was not enough matching metadata to resolve to a single one.
InjectionError The error that is thrown when an export could not satisfy an injection.

Function Description

lazyInjectMany(exportDecl) Lazily resolves and returns an array of exported values that all satisfy a given export declaration.
lazyInjectMany(exportDecl, metadata) Lazily resolves and returns an array of exported values that all satisfy a given export declaration.
