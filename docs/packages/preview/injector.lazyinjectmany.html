lazyInjectMany() function

Lazily resolves and returns an array of exported values that all satisfy a given export declaration.

Signature:

export declare function lazyInjectMany<T, TMetadata>(exportDecl: ExportDeclaration<T, TMetadata>): LazyExport<T, TMetadata>[];

Parameters

Parameter Type Description
exportDecl ExportDeclaration<T, TMetadata> The ExportDeclaration for which a value is required.

Returns:

LazyExport<T, TMetadata>[]

An array of LazyExport that can be used to obtain the resolved export satisfying exportDecl.
