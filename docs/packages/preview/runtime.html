runtime package

Provides the core runtime primitives that support all Converge.js packages.

Classes

Class Description
InvalidOperationError The error that is thrown when a method invocation is invalid for the current state of an object.
OperationAbortedError The error that is thrown when a method invocation has been aborted.
PromisedValueSource Represents the producer side of a Promise unbound to any particular source, providing access to the consumer side through the promise property.
RuntimeAssertionError An error that represents a failed runtime assertion.
RuntimeError Serves as the base class for runtime errors.
WaitHandle A coroutine based synchronization event.

Enumerations

Enumeration Description
ConsoleLoggerType Indicates a console logger.
RuntimeFlags Represents a collection of runtime flags.

Functions

Function Description
assert(value, message, assertOptions) Performs a runtime assertion.

Interfaces

Interface Description
AssertOptions Defines the options that can be used with assert().

Variables

Variable Description
CurrentRuntime The runtime flags of the currently executing process.
CurrentRuntimeGlobal The runtime global object.
