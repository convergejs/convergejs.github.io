Export.instance() function

Provides an export function that will return new instances of a given class type.

Signature:

function instance<T>(target: {
        new (...args: any[]): T;
    }): ExportFunction<T>;

Parameters

Parameter Type Description
target { new (...args: any[]): T; } The constructor of the class type to be exported.

Returns:

ExportFunction<T>

A function that will return a new instance of target when it is invoked.
